js 函数是否可以重载?

考虑到 js 函数调用时候的自由程度, 实际上 js 函数根本不需要重载, 就可以随便用

注意: js 中不能有重名的函数, 当有多个函数重名时, 后声明的函数会覆盖前面的函数

现在假设 有两个函数:
函数1:   function test1() { alert("没有参数的 tes1 t函数"); }
函数2:   function test1(username) { alert("有参数的 test1 函数"); }
