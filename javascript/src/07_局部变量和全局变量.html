全局变量和局部变量

全局变量

在函数体之外声明的变量是全局变量, 在浏览器打开时声明, 关闭时销毁 尽量少用全局变量, 会占用大量浏览器内存

全局变量会一直在浏览器的内存中, 耗费内存空间

局部变量

在函数体中声明的变量, 包括函数形参, 都属于局部变量

局部变量的声明周期, 函数开始执行时, 局部变量的内存空间开辟, 函数结束后, 局部变量的内存空间释放

局部变量声明周期较短

局部变量与全局变量同名

当局部变量和全局变量同名, 并在函数中调用时, 会采用就近原则


假设有全局变量:   var username = "jack";
然后有函数定义:   function accessUsername () { var username = "rose"; alert("user name is: " + username); }
那么调用函数后会优先采取 函数中的局部变量 rose

js 独一档语法:

如果一个变量在声明的时候没有使用 var 关键字, 那么这个变量不论在哪里声明, 都是全局变量

现在有函数声明如下:
function myFun () { myName = "Waynn"; }
然后先调用函数:
myFun();
此时函数内部声明的变量就是一个全局变量, 可以做输出:
alert("myName: " + myName);

如果现在把函数定义中的变量添加 var 关键字, 那么会因为他变成一个局部变量导致报错.
