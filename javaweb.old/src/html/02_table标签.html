这个html用来介绍 table 标签

表格用的是 table 标签, table 标签下面,

  • th 表示表头
  • tr 表示行
  • td 表示列
  • 可以用 rowspan 和 colspan 表示行合并或者列合并


以下几个属性目前已经被淘汰, 所以了解一下即可

  • 在 table 标签下添加 border 属性, 即可为表格添加边框

  • 可以用 width 和 height 设置宽高

  • 通过 cellspacing 属性设置单元格间距

  • 通过 cellpading 属性 可以设置每一个单元格内容文字和 边框的距离

  • 在 tr 标签内使用 align 标签, 可以设置文字对齐方式
