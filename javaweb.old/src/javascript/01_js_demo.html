javascript 基本语法

基本介绍

javascript 是客户端脚本语言, 语法风格和 java 很像. 使用 'script' 标签创建内部 javascript 代码环境.
javascript 也包括基本数据类型和 引用数据类型, 其中数值型不区分整数和小数, 字符串 不区分字符, 字符串; 单双引号作用相同.

布尔值比较特殊:
非零数值, 非空字符串, 非空对象, 转换成 bool 值, 都是 true.
相反, 0, 空字符串, null, undefined 都是 false.

引用数据类型

引用类型包括:
  • new 出来的对象
  • 用 [] 声明的数组
  • 用 {} 声明的对象

javascript 函数

javascript 的函数语法十分自由,
定义函数的时候, 不需要定义访问权限修饰符, 不需要定义返回值类型, 函数可以 return 也可以不 return,
输入实参的时候可以根据函数定义的形参列表输入 也可以只输入一部分甚至不输入参数.
